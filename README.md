# personal-finance-manager
Finance Manager

A full-stack Finance Management application built using Spring Boot.

Features
Add income and expense transactions
Track financial records
REST API integration
CRUD operations
Transaction categorization
Backend validation and exception handling

Java
Spring Boot
Maven
REST API
JPA/Hibernate
Database
MySQL

Create Transaction
Update Transaction
Delete Transaction
Get All Transactions
Filter Transactions
