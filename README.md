# 💰 Finance Manager

A backend Finance Management application built using Java and Spring Boot that helps users manage income and expense transactions efficiently through REST APIs.

---

## 🚀 Features

- Add income and expense transactions
- Update and delete financial records
- View all transactions
- RESTful API architecture
- Layered backend structure
- Exception handling and validation
- MySQL database integration

---

## 🛠️ Tech Stack

### Backend
- Java
- Spring Boot
- Maven
- REST API
- JPA / Hibernate

### Database
- MySQL
